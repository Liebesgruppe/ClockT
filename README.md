# ClockT

# 多功能时钟

### 简介

包含常用的闹钟、时间、秒表、计时器四部分。

*   主界面通过viewpager+fragment实现

*   view动画、XML绘图、SVG矢量动画

*   BroadcastReceive+Service实现闹的唤醒

*   AlarmManager、TImer、TimerTask的使用
